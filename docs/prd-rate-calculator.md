# PRD: 费率计算器

## Problem Statement

报关员日常高频操作：查 HS 编码的税率 → 手动计算关税、增值税、消费税 → 估算综合税费。当前流程需要打开多个网站/文档查询税率，再用计算器手工算，效率低且易出错。

## Solution

新增独立模块「费率计算器」，输入 HS 编码 + 原产国 + 货值 → 一键计算综合税费。结果清晰展示分项明细和合计，历史记录可回溯。

## User Stories

1. 作为报关员，我希望输入 HS 编码后自动显示适用税率，以便无需手动查税率表。
2. 作为报关员，我希望输入货值和数量后自动计算关税金额，以便快速得到税费结果。
3. 作为报关员，我希望同时看到增值税和消费税的计算结果，以便全面了解综合税负。
4. 作为报关员，我希望选择原产国后，系统自动判断适用最惠国税率还是普通税率，以便正确计算关税。
5. 作为报关员，我希望查看历史计算记录，以便回溯之前的计算结果。
6. 作为报关员，我希望从归类模块直接跳转到计算器并带入 HS 编码，以便无缝衔接工作流。
7. 作为报关员，我希望计算器界面简洁清晰，输入和结果分区明确，以便快速上手使用。

## 计算公式

**关税** = 完税价格(CIF) × 关税税率
- 最惠国税率（MFN）：适用于 WTO 成员国
- 普通税率：适用于非 WTO 成员国
- 暂定税率：部分商品适用，优先于 MFN

**增值税** = (完税价格 + 关税) × 增值税率
- 一般为 13%，部分商品 9%

**消费税**（如适用）= (完税价格 + 关税) / (1 - 消费税率) × 消费税率
- 仅特定商品（烟酒、化妆品、汽车等）征收

**综合税费** = 关税 + 增值税 + 消费税

| 元素 | 行为 |
|---|---|
| HS 编码输入 | 输入时实时校验格式(10位数字+点号)，回车触发计算 |
| 原产国下拉 | 从 Settings > 基础数据 > 国家/地区读取 |
| 完税价格 | 数字输入，自动千分位格式化显示 |
| 计算按钮 | 主按钮主题色，计算中显示 loading 态，完成后变"重新计算" |
| 历史项点击 | 回填输入参数 + 展开历史结果 |
| NavRail 导航 | 新增计算器图标(🖩)，与制单/归类/设置并列 |

## Implementation Decisions

### 税率数据来源

解析已有税则文件 `prompts/china-hscode-rule-2026.md`。格式示例：

```
8414.1000 真空泵
  最惠国税率: 8%
  普通税率: 30%
  增值税率: 13%
  监管条件: A
  法定单位: 台/千克
```

使用 IPC 在后台解析税则文件，构建 HS 编码 → 税率信息的查找表。解析策略：
- 读取税则文件，按行解析
- 匹配 HS 编码行（前 10 位为数字和点号）
- 提取后续行的税率信息
- 缓存到 Map 中

### 原产国税率判断

- 默认适用最惠国税率（WTO 成员国占绝大多数）
- 用户手动选择"普通税率"时切换
- 暂定税率优先于最惠国税率（如税则中有标注）

### 数据库

新建 `calculator_history` 表：

- `id` TEXT PRIMARY KEY
- `hs_code` TEXT
- `hs_description` TEXT（商品描述，来自税则）
- `country_code` TEXT（原产国代码）
- `cif_value` REAL
- `quantity` REAL
- `duty_rate` REAL（实际使用的关税税率）
- `duty_amount` REAL
- `vat_rate` REAL
- `vat_amount` REAL
- `consumption_tax_rate` REAL
- `consumption_tax_amount` REAL
- `total_tax` REAL
- `total_price` REAL
- `result_json` TEXT（完整结果 JSON）
- `created_at` TEXT

### IPC 通道

- `calculator:lookup` — 查询 HS 编码的税率信息
- `calculator:history-list` — 获取历史记录
- `calculator:history-save` — 保存计算记录

### NavRail 导航

NavRail 底部增加「计算器」图标（可新增 IconCalcNav），与设置按钮并列。

### 弃用 tax_category 字段手动输入

币制和国家已从基础数据中选择，不在计算器页手动输入。

## Testing Decisions

- 测试税率解析逻辑（从税则文件中正确提取税率）
- 测试计算器核心公式（关税/增值税/消费税）
- 不测试：税则文件格式变更（属于数据层）

## Out of Scope

- 跨境电商综合税率计算
- 出口退税计算
- 汇率自动换算
- 多 HS 编码批量计算
- 打印/导出计算结果
