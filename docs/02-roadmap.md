## 产品哲学

1. **一个版本只解决一个核心问题**。不给用户堆功能。
2. **每个版本有独立的用户价值**。不需要等全部做完才能用。
3. **模块可组合**。用哪个开哪个，不用的不占界面空间。
4. **先纵深后横向**。先把一条完整链路做透（提取→编码→审核→输出），再扩展到业务管理。

## 产品全景

```
                    DeclarAI
                        
  Extract ──→ Smart Code ──→ Verify ──→ Output
  智能提取      智能编码       专业审核      模板输出
  v1.0 ✅                   
                              
              ┌──── 共享数据层（通用字段池 + 文件存储）────┐
              
                         Desk 业务工作台
                         客户·状态·历史
                            v5.0
```

五个能力模块，通过共享数据层自然衔接。Extract+Code+Verify+Output 构成核心链路，Desk 是业务管理层。

## v2.0 Smart Code（智能编码）

**核心问题**：AI 提取出中文描述，但报关单要填海关标准代码。报关员每天花最多时间查 HS 编码。

**依赖关系**：2.2（HS 推荐）→ 2.3（申报要素）依赖 HS 编码确定后才能提示要素。

### 2.1 海关代码表

内置 8 类海关标准代码表（静态 JSON，`src/main/code-tables/`）。select 字段值从 `string` 升级为 `{ label: string, code: string }`。UI 显示 label，存储同时存 code。类型 registry 的 select options 改为从代码表动态读取。

范围：监管方式、征免性质、成交方式、运输方式、币制、关区、包装种类、港口。

### 2.2 HS 编码推荐

商品名称 → AI 推荐 HS 编码（含 4/6/10 位建议）。推荐附带：法定单位、关税税率参考、增值税率参考、监管条件（许可证等）。

实现：AI prompt 注入常用章节 HS 知识。不做完整税则数据库。后续可接入专业归类 API。

### 2.3 申报要素补全 **[依赖 2.2]**

确定 HS 编码后，提示该编码对应的必填申报要素（品牌、型号、材质、用途、成分含量等）。缺失项在货物明细行中高亮。

## v3.0 Verify（专业审核）

**核心问题**：AI 填完数据后需要报关专业知识审核，而非通用"有没有填"检查。

### 3.1 单证交叉一致性

提取阶段为每个字段打来源标记（来自发票/箱单/提单/合同）。审核阶段对比不同来源的同一字段：金额、件数、重量一致性。不依赖 AI，确定性规则。

### 3.2 报关逻辑校验

规则引擎硬编码：净重≤毛重、总价≈单价×数量、CIF≥FOB、成交方式与运费逻辑、HS 与商品名合理性、港口与运输方式一致性。

模糊项（HS 归类合理性、价格是否异常）由 AI 辅助判断。

### 3.3 单证完整性检查

根据贸易方式判断应有哪些单证。一般贸易进口须有合同+发票+装箱单+提单。法检商品须有通关单。缺单证给提醒。

## v4.0 Output（模板输出）

**核心问题**：数据在软件里，报关员需要"搬"到单一窗口。输出格式应匹配单一窗口录入顺序。

### 4.1 抄录视图

按每种单据在单一窗口中的字段排列生成抄录视图。字段按页面分组（表头→表体→表尾），显示代码值+中文。可一键复制单个字段值。

### 4.2 XLSX 导出

每种单据一个 Excel 模板配置（SheetJS），字段映射到单元格。支持批量导出。自动留存到 `exports/` 目录。

---

## v5.0 Desk（业务工作台）

**核心问题**：报关员管理多客户多票业务，需要超越单票视角。

### 5.1 客户维度

申报单关联客户。按客户筛选、分组。客户信息可设为字段默认值。最小化实现，不做 CRM。

### 5.2 申报状态流

真实状态机：草稿→待申报→已申报→海关审核中→退回补证→已修改→重新申报→放行。状态变更日志。

### 5.3 工作台

看板视图（按状态分栏），本周待处理/被退单/超期数。按客户、时间筛选。

### 5.4 历史复用

新建申报单可基于历史创建。同客户同品名自动建议复用上一票数据。

---

## 不做

单一窗口 API 直连、关税税则完整数据库、在线支付、多用户协作/云端同步、OCR 图片 PDF、移动端 App、物流追踪、货代管理、财务结算。
