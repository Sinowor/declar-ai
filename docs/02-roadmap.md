# DeclarAI 产品演进路线图

> 更新: 2026-06-06

## 产品哲学

1. **一个版本只解决一个核心问题**。不给用户堆功能。
2. **每个版本有独立的用户价值**。
3. **模块可组合**。用哪个开哪个。

---

DeclarAI

## v2.5 费率计算器（新模块：calculator）

**问题**：报关员需手动查税率→计算关税+增值税+消费税→估算成本。效率低。

**核心功能**：
- 输入 HS 编码 + 原产国 + 货值 + 数量 → 一键出综合税费
- 关税 = 货值 × 税率（从税则文件解析）
- 增值税 = (货值 + 关税) × 增值税率
- 消费税 = (货值 + 关税) / (1 - 消费税率) × 消费税率（如适用）
- 结果卡片：分项明细 + 合计 + 完税总价
- 历史计算记录保存

**界面**：左侧输入 → 右侧结果卡片。可从归类模块跳转带入 HS 编码。

**数据源**：已有 `china-hscode-rule-2026.md` 税则文件，税率可从中解析。

## v2.6 知识库（新模块：knowledge）

**问题**：归类经验、客户特殊要求、口岸注意事项无法沉淀复用。

**核心功能**：
- 条目：标题 + HS 编码（可选关联）+ 标签 + Markdown 正文
- 标签：归类经验 / 客户备注 / 口岸须知 / 操作流程 / 自定义
- 搜索 + 标签过滤
- 与归类模块集成：HS 编码匹配时展示相关知识条目

**存储**：SQLite 本地表，Markdown 存储正文。

**界面**：左侧标签过滤 + 列表，右侧正文渲染。简洁备忘录风格。

## v3.0 制单内化：合规检查 + 审核工作流

两个功能都内化到现有的制单 AI 审核环节（Block ②）。

### 3.1 合规检查

在 Review Issues 中增加「合规风险」类别：

| 检查项 | 说明 |
|---|---|
| 许可证要求 | 该 HS 编码是否需进口/出口许可证 |
| 禁限商品 | 是否属于禁止/限制进出口目录 |
| 原产地证 | 是否适用优惠税率需提供原产地证 |
| 反倾销 | 原产国是否涉及反倾销税 |
| 检疫要求 | 是否需要动植物检疫/食品检验 |

高风险项红色提醒，需人工确认。数据来自税则文件中的监管条件字段。

### 3.2 审核工作流

状态流扩展：

```
草稿 → AI提取中 → 待审核 → 已审核 → 已完成
                    ↑         │
                    └─── 驳回 ──┘
```

- 审核人确认（姓名 + 日期）
- AI 提取值 vs 人工修改值字段高亮对比
- 审核意见备注 + 驳回原因记录

### v7.0 RAG 智能知识库（远期规划）

**核心能力**：用户拖入海关公告/法规/单证模板 → AI 自动提取、分块、打标签、向量化存储。后续通过对话提问，AI 基于知识库内容回答并引用来源。

**技术栈**：

| 组件 | 方案 | 说明 |
|---|---|---|
| 文档解析 | 已有（PDF/Word/Excel 提取） | 复用现有 extractor |
| 文本分块 | LangChain.js `RecursiveCharacterTextSplitter` | 按语义边界分块 |
| Embedding | `transformers.js` + `all-MiniLM-L6-v2` | 纯本地，不需要 GPU |
| 向量存储 | `sqlite-vec` | SQLite 扩展，与现有 DB 统一 |
| 检索 | 语义相似度 + BM25 混合 | 精确 + 模糊 |
| 回答生成 | 现有 DeepSeek API | 带上下文 prompt |

**开源参考**：
- [sqlite-vec](https://github.com/asg017/sqlite-vec) — SQLite 向量扩展，零配置
- [transformers.js](https://github.com/xenova/transformers.js) — HuggingFace 模型在 Node.js 运行
- [LangChain.js](https://github.com/langchain-ai/langchainjs) — RAG 流程编排
- [NVIDIA GenerativeAIExamples](https://github.com/NVIDIA/GenerativeAIExamples) — 企业级 RAG 参考架构

**交互**：
- 拖拽文件入库（PDF/Word/MD/图片）
- AI 自动总结、打标签、分类
- 对话式提问：「二连口岸最近有什么新规定？」
- 回答引用来源文档 + 片段高亮
- 手动笔记入口保留（轻量 Markdown）
