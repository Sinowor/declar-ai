## 全局导航：左侧图标导航栏

废弃侧边栏顶部 Tab 切换方案，改用**左侧图标导航栏**（Icon Rail）。这是桌面工具类应用的标准导航模式。

```
┌────┬──────────────┬──────────────────────────────────────────┐
│ 📋 │              │                                          │
│    │   上下文面板   │              工作区                      │
│ 🔍 │   (280px)    │             (flex-1)                     │
│    │              │                                          │
│ 📊 │  随导航图标   │    随导航图标切换而切换                     │
│    │  切换而切换    │                                          │
│ ⚙  │              │                                          │
│    │              │                                          │
└────┴──────────────┴──────────────────────────────────────────┘
 48px     280px                    flex-1
 导航栏   上下文面板                主工作区
```

### 导航栏规格

- 宽度：48px（折叠态）/ 可无折叠，始终窄条
- 图标：20×20px SVG，上下排列
- 间距：图标之间 8px，首个图标距顶 12px
- 选中态：图标左侧 3px 紫色竖线 + 图标颜色变为 primary
- 未选中态：灰色图标，hover 变深灰
- 底部固定：设置图标（⚙）始终在最底部

### 导航项

| 图标 | 模块 ID | 模块名称 | 上下文面板内容 | 工作区内容 | 状态 |
|------|--------|---------|---------------|-----------|------|
| 📋 | `declarations` | 申报制单 | 申报单列表 + 搜索 + 新建 | 三叠式编辑界面 | ✅ 已有 |
| 🔍 | `hs-classifier` | HS 归类 | 历史记录 + 搜索 + 新建 | 归类咨询（三态） | 本次新增 |
| 📊 | `desk` | 工作台 | 状态筛选 + 客户列表 | 看板仪表盘 | v5.0 规划 |
| ⚙ | `settings` | 设置 | 配置项列表 | 设置表单 | 待设计 |

### 导航切换行为

- 点击图标 → 上下文面板 + 工作区同时切换到对应模块
- 保留各自滚动位置和输入状态（`state-preservation`）
- 默认打开上次关闭时的模块
- 无动画跳变（即时切换），不搞花哨过渡

## 与申报制单的联动

在申报制单模块的货物明细表中点击 `[🔍 HS归类]`：

1. 自动切换到 HS 归类模块
2. 货物名称 + 规格型号预填入输入框
3. 自动触发归类分析
4. 结果面板增加「应用到申报单」按钮
5. 点击后 HS 编码写入该货物行的 `hs_code` 字段，切回申报制单模块

## 数据存储

### DB 表

```sql
CREATE TABLE hs_classifications (
  id TEXT PRIMARY KEY,
  product_description TEXT NOT NULL,
  hs_code TEXT,
  hs_description TEXT,
  confidence TEXT,
  mfn_rate TEXT,
  vat_rate TEXT,
  supervision_conditions TEXT,
  full_result_json TEXT NOT NULL,
  report_path TEXT,
  created_at TEXT NOT NULL DEFAULT (datetime('now','localtime'))
);
```

### 报告文件

`~/Documents/DeclarAI/HS归类/[商品名]-[HS编码]-归类报告.md`

## UX 规则遵循

| 规则 | 应用 |
|------|------|
| `adaptive-navigation` | 桌面端≥1024px 使用图标导航栏 |
| `nav-label-icon` | 图标有 tooltip 显示模块名称 |
| `nav-state-active` | 选中图标有紫色竖线 + primary 颜色 |
| `state-preservation` | 切换模块保留各自的滚动位置和输入状态 |
| `primary-action` | 每屏只有一个主按钮 |
| `empty-states` | 空状态有引导说明 + 快捷示例 |
| `loading-states` | 分析中显示步骤进度 |
| `progressive-disclosure` | 归类依据默认折叠 |
