## Phase 1.5c: 存储验证 ✅

### Acceptance criteria

- [x] 新建申报单后，对应文件夹创建（如 `D000001-(未命名)/`）
  - `declaration:create` → `folderPath(seqNo, displayName)` → `ensureFolder(fPath)`
- [x] 导入文件后，文件副本出现在 `files/` 子目录
  - `file:import` → `importFileToFolder(folder, srcPath)` → path.join(filesDir, ...)
- [x] AI 提取后 `declaration.json` 包含完整数据
  - `extractor.ts` → `writeJsonFile(declarationJsonPath(row.folder_path), extractedData)`
- [x] 人工编辑保存后，`declaration.json` 数据更新
  - `declaration:update` → `writeJsonFile(declarationJsonPath(row.folder_path), sanitized)`
- [x] 删除申报单后，对应文件夹被删除
  - `declaration:delete` → `deleteFolder(row.folder_path)`
- [x] 旧版 userData 存储被自动清理
  - `main/index.ts initApp()` → unlink oldDB + rmSync oldFiles
